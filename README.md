CAS Kozhikode Website

A static single-file website for **College of Applied Science Kozhikode**, an institution under IHRD, Government of Kerala.

---

## About

This is the website presence for CAS Kozhikode. Built as a single self-contained HTML file — no build tools, no frameworks, no dependencies.

**Built with:** HTML · CSS · Vanilla JavaScript  
**Fonts:** Google Fonts (Merriweather + Source Sans 3)

---

## Source

The original HTML was taken from the live IHRD-hosted site and reworked from scratch — fixing responsiveness, desktop layout, container overflow, broken JavaScript, and overall structure — while keeping all the original content, links, and design intact.

---

## Features

- Fully responsive — works on mobile, tablet, and desktop
- Sticky header with dropdown navigation
- Hero section with college details
- Course listings, institution grid, and quick links
- College info, contact details, and footer
- Slide-in mobile navigation
